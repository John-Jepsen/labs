# Absolute Beginner Setup

This section will guide you through setting up your coding environment from scratch. If you've never written code before, don't worry! We'll walk through each step with clear instructions and images.

## What You'll Need

- A computer running Windows, macOS, or Linux
- Internet connection
- About 30-60 minutes of time

## Setup Steps

1. [Installing Visual Studio Code](install_vscode.md)
2. [Installing Python](install_python.md)
3. [Installing Git](install_git.md)
4. [Installing Node.js](install_nodejs.md)
5. [Creating Your First Python Program](first_python.md)
6. [Running Code in the Terminal](terminal_basics.md)

After completing the setup, you'll be ready to start learning how to code with your AI tutor! The next steps will be to:
